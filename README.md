В системе реализовано:
- учет клиентов и оборудования
- регистрация заявок на обслуживание
- учет материалов и работ
- бухгалтерские проводки по операциям
- отчеты по остаткам и задолженностям



## Функционал
- создание заявки на обслуживание
- списание материалов
- поступление материалов от поставщиков
- учет оплат клиентов
- формирование отчетов
- формирование печатных форм
